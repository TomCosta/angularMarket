#### Instalar Angular:
```javascript
npm install @angular/cli -g
``` 
#### Clonar o repositório:
```javascript
git clone https://github.com/TomCosta/angularMarket.git
``` 
#### Instalar as dependências:
```javascript
npm install
``` 
#### Rodar o projeto localmente:
```javascript
ng serve
```
